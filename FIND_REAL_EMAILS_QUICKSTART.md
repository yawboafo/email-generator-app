## üö® Troubleshooting

### "No emails found"
- **Web Scraping**: Try different page (like /contact or /about)
- **APIs**: Check spelling, try different search terms

### "Invalid API key"
- Copy key again carefully (no extra spaces)
- Verify account is active
- Check you're using correct API provider

### "Rate limit exceeded"
- Wait 1 hour
- Switch to different method
- Consider paid tier for higher limits

### "Website blocked request"
- Some sites block scrapers
- Try API methods instead
- Look for alternative pages on same site
